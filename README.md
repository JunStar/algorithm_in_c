## sort 排序

### BubbleSort 冒泡排序
### InsertSort - Direct Insert Sort 插入排序(直接插入排序)
### ShellSort 希尔排序
### BinInsertSort 二分查找插入排序

> 直接插入排序对于大部分有序的序列排序时速度快。
> 二分插入排序在直接插入的基本上改变了查找元素插入位置的方法，对于完全无序的序列来说，速度会变快，但是对开大部分有序的序列反而会更慢。
> 希尔排序则利用直接插入排序对于大部分有序的序列速度快的特点，先让大部分序列有序，以此来提高排序效率。
